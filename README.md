# 🎓 GosProgram MVP — Генератор программы ДПО государственного образца

> Мультиагентный Claude Code skill для автоматической генерации программ дополнительного профессионального образования (ДПО) по эталонам государственного образца.

## 🎯 Задача

Методисты онлайн-университета Zerocoder создают программы ДПО для каждого нового курса. Процесс требует:
- Извлечения данных курса из Notion
- Приведения структуры в соответствие с ГОСТ-требованиями
- Генерации 13 разделов программы по эталонным шаблонам
- Расчёта академических часов, учебных планов, календарных графиков
- Оформления в формате .md и .docx (Times New Roman, 14pt, ГОСТ-поля)

Ручная генерация занимает **3–5 часов**. Этот skill сокращает время до **15–30 минут** с итеративным уточнением.

## 🏗️ Архитектура

Мультиагентный пайплайн из 4 этапов:

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  Extractor   │───▶│  Analyzer   │───▶│   Writer    │───▶│  Formatter  │
│  (Notion →   │    │  (Сравнение │    │  (Генерация │    │  (.md +     │
│   JSON)      │    │   с эталоном)│    │   программы)│    │   .docx)    │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

| Агент | Задача | Инструменты |
|-------|--------|-------------|
| **Extractor** | Читает Notion-страницу, извлекает структуру курса | Notion API, Python |
| **Analyzer** | Сравнивает данные с эталонами, выявляет пробелы | LLM, эталонные программы |
| **Writer** | Генерирует компетенции, ЗУН, задачи, описания | LLM, Jinja2-шаблоны |
| **Formatter** | Собирает финальный документ, экспортирует | python-docx |

## 🚀 Установка и запуск

### Предварительные требования

- Python 3.10+
- Claude Code CLI
- Notion API-токен

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Настройка Notion API

Создайте файл `.env` в корне проекта:

```env
NOTION_API_KEY=your_notion_integration_token_here
```

### Запуск через Claude Code

```bash
# В Claude Code терминале:
/gosprogram https://www.notion.so/zerocoder/Course-Page-123

# С дополнительными параметрами:
/gosprogram https://www.notion.so/zerocoder/Course-Page-123
Название: Промпт-инжиниринг 3.0
Автор: Иванов Иван
Профстандарт: Системный аналитик
```

### Ручной запуск (без Claude Code)

```bash
# Извлечение данных из Notion
python src/notion_extractor.py "https://www.notion.so/..." --output output/course_data.json

# Экспорт в DOCX (после генерации .md)
python src/docx_export.py --input output/program_dpo.md --output output/program_dpo.docx
```

## 📋 Разделы программы

Генерируемая программа содержит 13 разделов в соответствии с требованиями ДПО:

1. **Общие положения** — нормативные документы, профстандарт, характеристика программы
2. **Цель и задачи программы** — цель и задачи в формате «заголовок: описание»
3. **Планируемые результаты обучения** — ОК, УК, ПК с кодами и описаниями
4. **Учебный план** — таблица модулей с часами (всего, лекции, практика, контроль)
5. **Календарный учебный график** — расписание по неделям
6. **Учебно-тематический план** — таблица уроков с часами и формами контроля
7. **Содержание курса** — описание каждого модуля и урока
8. **Организационно-педагогические условия** — ДOT, расписание, обратная связь
9. **Оценка качества освоения программы** — система оценивания, сертификаты
10. **Контрольно-оценочные материалы** — тесты, практические задания
11. **Материально-технические условия** — оборудование, ПО
12. **Информационные и учебно-методические условия** — литература, ресурсы
13. **Титульный лист** — организация, ФИО директора, автор, год

## 🔧 Технологии

- **Claude Code** — мультиагентная оркестрация, генерация текста
- **Notion API** — извлечение структуры курса (модули, уроки, часы)
- **python-docx** — генерация DOCX с форматированием по ГОСТу
- **Jinja2-шаблоны** — форматирование разделов программы
